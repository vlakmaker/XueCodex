📊 Computing Derivatives with a Computation Graph

This notebook explains how to compute derivatives using a computation graph, following the example from Andrew Ng's neural networks course. It breaks down a simple function into smaller steps and shows how to apply the chain rule to calculate how changes in inputs affect the final output.

🧠 The Function​

We are given:

J=3(a+bc)J = 3(a + bc)J=3(a+bc)

Let’s break it into smaller steps:

  • u=b×cu = b \times cu=b×c
  • v=a+uv = a + uv=a+u
  • J=3×vJ = 3 \times vJ=3×v

✅ Forward Pass​

Given values:

  • a=5a = 5a=5
  • b=3b = 3b=3
  • c=2c = 2c=2

We compute:

  • u=3×2=6u = 3 \times 2 = 6u=3×2=6
  • v=5+6=11v = 5 + 6 = 11v=5+6=11
  • J=3×11=33J = 3 \times 11 = 33J=3×11=33

🔁 Backward Pass (Derivatives)​

We use the chain rule to compute how much changing each input affects the output JJJ.

Step 1: dJdv=3\frac{dJ}{dv} = 3dvdJ​=3​

Because:

J=3v⇒If v↑1,J↑3J = 3v \Rightarrow \text{If } v \uparrow 1, J \uparrow 3J=3v⇒If v↑1,J↑3

Step 2: dvdu=1\frac{dv}{du} = 1dudv​=1​

Because:

v=a+u⇒If u↑1,v↑1v = a + u \Rightarrow \text{If } u \uparrow 1, v \uparrow 1v=a+u⇒If u↑1,v↑1

Step 3: dJdu=dJdv×dvdu=3×1=3\frac{dJ}{du} = \frac{dJ}{dv} \times \frac{dv}{du} = 3 \times 1 = 3dudJ​=dvdJ​×dudv​=3×1=3​

📦 Derivatives with Respect to Inputs​

1. dJda\frac{dJ}{da}dadJ​​

  • v=a+u⇒dvda=1v = a + u \Rightarrow \frac{dv}{da} = 1v=a+u⇒dadv​=1
  • dJda=dJdv×dvda=3×1=3\frac{dJ}{da} = \frac{dJ}{dv} \times \frac{dv}{da} = 3 \times 1 = 3dadJ​=dvdJ​×dadv​=3×1=3

2. dJdb\frac{dJ}{db}dbdJ​​

  • u=b×c⇒dudb=c=2u = b \times c \Rightarrow \frac{du}{db} = c = 2u=b×c⇒dbdu​=c=2
  • dJdb=dJdu×dudb=3×2=6\frac{dJ}{db} = \frac{dJ}{du} \times \frac{du}{db} = 3 \times 2 = 6dbdJ​=dudJ​×dbdu​=3×2=6

3. dJdc\frac{dJ}{dc}dcdJ​​

  • u=b×c⇒dudc=b=3u = b \times c \Rightarrow \frac{du}{dc} = b = 3u=b×c⇒dcdu​=b=3
  • dJdc=dJdu×dudc=3×3=9\frac{dJ}{dc} = \frac{dJ}{du} \times \frac{du}{dc} = 3 \times 3 = 9dcdJ​=dudJ​×dcdu​=3×3=9

✅ Final Summary Table​

VariableDerivative
aaadJda=3\frac{dJ}{da} = 3dadJ​=3
bbbdJdb=6\frac{dJ}{db} = 6dbdJ​=6
cccdJdc=9\frac{dJ}{dc} = 9dcdJ​=9

This tells us how much each input influences the final output JJJ using derivatives and chain rule in a computation graph.
